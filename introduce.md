## function

function 是一個小工廠，依據你的輸入值去完成特定的行為或工作之後
回傳相對應的值或是不回傳值

##Function 重要的組成：

    * Input : 函數的輸入值

    * logic : 函數中的邏輯運算

    * output : 函數回傳值
